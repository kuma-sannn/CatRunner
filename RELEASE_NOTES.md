# GPU Cat Monitor v1.0.1

## 🐱 Features

- **System Tray Animation** - Cute pixel cat runs in your system tray
- **Real-time Monitoring** - GPU & CPU usage stats updated every 500ms
- **Floating Cat Window** - Draggable cat that floats on your desktop
- **Speed-based Animation** - Cat runs faster when GPU/CPU load increases
- **Hover Stats** - Detailed GPU/CPU info on hover

## 🖥️ System Requirements

- Windows 10 or Windows 11
- 64-bit processor

## 📥 Installation

1. Download `GpuCatMonitor.exe`
2. Double-click to run (no installation needed)
3. Cat appears in system tray!

## 🎮 How to Use

- **Tray Icon**: Shows animated cat + GPU/CPU stats on hover
- **Right-click Menu**: Show floating cat, toggle animation mode, exit
- **Floating Cat**: Draggable window with cat animation
- **Double-click Tray**: Show/hide floating cat
